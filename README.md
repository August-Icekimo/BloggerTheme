## English

Welcome to the repository for managing the Blogger theme XML configuration for [https://blog.icekimo.idv.tw/](https://blog.icekimo.idv.tw/). 

This repository serves as a version control and backup for the custom theme used on the blog, which currently utilizes the **Igniplex v3.1** template.

### File Structure
* `Igniplex v3.1.xml`: The core Blogger theme XML file containing all the layout, styling, and widget configurations.
* `Changelog.txt`: Original theme documentation tracking the update history and features added in the Igniplex theme.

### Usage
To apply this theme to a Blogger site:
1. Go to your Blogger Dashboard.
2. Navigate to **Theme** on the left menu.
3. Click the down arrow next to the "Customize" button and select **Restore**.
4. Click **Upload** and select the `Igniplex v3.1.xml` file.

*(Note: Always backup your current theme before applying a new one.)*

## 繁體中文

歡迎來到這個主題設定檔儲存庫。本儲存庫專門用來管理與備份個人網誌 [https://blog.icekimo.idv.tw/](https://blog.icekimo.idv.tw/) 所使用的 Blogger 主題 XML 設定。

目前網誌使用的版型為 **Igniplex v3.1**。

### 檔案結構
* `Igniplex v3.1.xml`: Blogger 主題的核心 XML 設定檔，包含所有版面配置、CSS 樣式與小工具設定。
* `Changelog.txt`: 主題的更新日誌，記錄了 Igniplex 版型的歷史更新與新增功能。

### 如何使用
若要將此主題套用至 Blogger：
1. 進入您的 Blogger 後台管理介面。
2. 點選左側選單的 **主題**。
3. 點擊「自訂」按鈕旁邊的向下箭頭，選擇 **還原**。
4. 點選 **上傳**，然後選擇本儲存庫中的 `Igniplex v3.1.xml` 檔案。

*(注意：在套用新主題之前，請務必先備份您目前的主題。)*
